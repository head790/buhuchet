Сравни налоговые режимы и дай рекомендацию по оптимизации.

Аргументы: $ARGUMENTS
Формат: прогнозируемый годовой доход (опционально). Без аргументов — расчёт на основе данных из journal.csv.

## Инструкции

1. Прочитай `data/profile.yaml` — текущий режим, регион, ОКВЭД, дата регистрации
2. Прочитай `data/journal.csv` — просуммируй доходы и расходы за последние 12 месяцев
3. Прочитай `data/clients.yaml` — определи долю юрлиц, ИП и физлиц в доходах
4. Если передан аргумент с суммой — используй как прогнозируемый годовой доход вместо данных журнала

## Расчёт по каждому режиму

### УСН 6% (доходы)

```
налог = доход * 6%
взносы_фикс = 53658 (2025) / 57752 (2026)
взносы_1пц = max(0, (доход - 300000) * 1%)
взносы = взносы_фикс + взносы_1пц
вычет = min(взносы_уплаченные, налог)  # 100% для ИП без сотрудников
итого = (налог - вычет) + взносы
```

### УСН 15% (доходы минус расходы)

```
база = доход - расходы
налог = база * 15%
мин_налог = доход * 1%
налог = max(налог, мин_налог)
взносы = взносы_фикс + взносы_1пц  # база для 1% = доходы, НЕ доходы-расходы
итого = налог + взносы
# Примечание: взносы включаются в расходы для расчёта базы
```

### Патент (ПСН)

```
# Стоимость патента зависит от региона и ОКВЭД
# Инструкция: посмотри стоимость на https://patent.nalog.gov.ru/info/
# для ОКВЭД и региона из profile.yaml
# Если не можешь определить точную стоимость — попроси пользователя указать

патент = стоимость_патента_за_год
вычет = min(взносы_уплаченные, патент)  # 100% для ИП без сотрудников
итого = (патент - вычет) + взносы

# Лимиты:
# Доход по патенту не должен превышать 60 млн руб./год
# Не более 15 сотрудников
```

### НПД (самозанятость)

```
налог = доход_от_физлиц * 4% + доход_от_юрлиц_и_ип * 6%
# Бонус 10 000 руб. (пожизненный) уменьшает ставки до 3% / 4%
взносы = 0  # не обязательны
итого = налог

# Лимиты:
# Доход не более 2 400 000 руб./год
# Нельзя иметь сотрудников
# Нельзя перепродавать товары
```

### Патент + УСН 6% (совмещение)

```
# Доход до потенциального дохода по патенту — покрывается патентом
# Остаток — облагается по УСН 6%

доход_патент = потенциальный_доход_по_патенту
доход_усн = max(0, доход - доход_патент)

патент = стоимость_патента
усн = доход_усн * 6%

# Взносы можно распределить между режимами пропорционально доходу
# или полностью вычесть из патента (если хватает)
вычет_патент = min(взносы, патент)
остаток_взносов = взносы - вычет_патент
вычет_усн = min(остаток_взносов, усн)

итого = (патент - вычет_патент) + (усн - вычет_усн) + взносы
```

## Точки перелома (рассчитать)

Для каждой пары режимов найди доход, при котором один становится выгоднее другого:

- УСН 6% vs НПД: при каком доходе НПД дешевле (с учётом лимита 2.4М)?
- УСН 6% vs Патент: при каком доходе патент дешевле?
- УСН 6% vs Патент+УСН: при каком доходе совмещение выгоднее?
- УСН 6% vs УСН 15%: при какой доле расходов 15% выгоднее? (обычно >60%)

## Шаги перехода между режимами

### С УСН на Патент (или Патент+УСН):
1. Подать заявление на патент (форма 26.5-1) за 10 рабочих дней до начала
2. Патент можно оформить с любой даты, не только с 1 января
3. При совмещении: вести раздельный учёт доходов

### С УСН на НПД:
1. Зарегистрироваться в приложении «Мой налог»
2. В течение 30 дней подать уведомление об отказе от УСН
3. Помнить о лимите 2.4М

### С НПД на УСН:
1. Подать уведомление о переходе на УСН до 31 декабря
2. Переход с 1 января следующего года
3. При превышении лимита НПД — автоматический переход, уведомить в течение 20 дней

## Неналоговые факторы

Обязательно упомяни в рекомендации:

| Фактор | УСН | Патент | НПД |
|--------|-----|--------|-----|
| Пенсионный стаж | Да (взносы обязательны) | Да | Нет (добровольно) |
| Кредитоспособность | Высокая | Высокая | Средняя |
| Найм сотрудников | Да | До 15 чел. | Нет |
| Админ. нагрузка | Средняя (декларация) | Низкая (нет декларации) | Минимальная |
| Работа с юрлицами | Без ограничений | Без ограничений | Есть нюансы |

## Предупреждение

В конце всегда добавляй:
"⚠️ Расчёт носит оценочный характер. Стоимость патента зависит от региона и ОКВЭД — проверьте на patent.nalog.gov.ru. Перед сменой режима проконсультируйтесь с налоговым специалистом. Сроки и правила перехода могут меняться."
