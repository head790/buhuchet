Покажи реальную налоговую нагрузку — эффективную ставку и сравнение.

Аргументы: $ARGUMENTS
Формат: период (опционально). Примеры: "Q1 2025", "2025". Без аргументов — последние 12 месяцев.

## Инструкции

1. Прочитай `data/profile.yaml` — налоговый режим
2. Прочитай `data/journal.csv` — доходы и расходы за период
3. Прочитай `data/payments.yaml` — уплаченные налоги и взносы за период

## Расчёт

```
доход = сумма income за период
расходы = сумма expense за период
валовая_прибыль = доход - расходы

налог = расчёт по режиму
взносы = фикс + 1% сверх 300К (пропорционально периоду)
итого_государству = налог + взносы

эффективная_ставка_от_дохода = итого_государству / доход * 100%
эффективная_ставка_от_прибыли = итого_государству / валовая_прибыль * 100%
```

## Бенчмарки для сравнения

```
# Работник по найму с аналогичной "прибылью" (зарплата на руки):
# Работодатель платит: зарплата * 1.3 (взносы 30%) + НДФЛ 13%
# Эффективная нагрузка на фонд оплаты: ~43%
# Для работника: получает ~67% от стоимости для работодателя

# ООО на ОСНО:
# НДС 20% + налог на прибыль 20%
# Эффективно: ~35-40% при выводе дивидендов

# ООО на УСН 6%:
# Те же 6% + взносы за директора + 13% НДФЛ с дивидендов
# Эффективно: ~20-25%

# Самозанятый (НПД):
# 4-6%, без взносов
# Эффективно: 4-6%
```
