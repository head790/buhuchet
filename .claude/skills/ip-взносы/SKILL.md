Рассчитай страховые взносы ИП.

Аргументы: $ARGUMENTS
Формат: год (опционально). Без аргументов — текущий год.

## Инструкции

1. Прочитай `data/profile.yaml` — дата регистрации, налоговый режим
2. Прочитай `data/journal.csv` — просуммируй доходы за год
3. Прочитай `data/payments.yaml` — найди уже уплаченные взносы

## Ставки страховых взносов

С 2023 года ИП платит единый фиксированный взнос (ОПС + ОМС объединены):

| Год  | Фиксированная часть | 1% свыше 300К | Максимум всего |
|------|---------------------|---------------|----------------|
| 2025 | 53 658 руб.         | 1% от (доход − 300 000) | 300 888 руб. |
| 2026 | 57 752 руб. (проект.) | 1% от (доход − 300 000) | ~330 000 руб. |

## Расчёт

### Фиксированная часть

```
если ИП зарегистрирован полный год:
    фикс = ставка_года (53658 для 2025)

если ИП зарегистрирован не с начала года:
    полных_месяцев = количество полных месяцев работы
    дней_в_неполном_месяце = дни в месяце регистрации
    фикс = (ставка / 12 × полных_месяцев) + (ставка / 12 / дней_в_месяце × дней_работы)
```

### Переменная часть (1% свыше 300К)

```
если годовой_доход > 300 000:
    переменная = (годовой_доход − 300 000) × 1%
    переменная = min(переменная, максимум − фикс)
иначе:
    переменная = 0

# Для УСН 15%: база для 1% = доходы (НЕ доходы минус расходы) — по текущей практике
# Для патента: база = потенциальный доход по патенту
# Для НПД: обязательных взносов НЕТ (добровольные)
```

## Сроки уплаты

- Фиксированная часть: **до 31 декабря** текущего года
- 1% свыше 300К: **до 1 июля** следующего года
- Рекомендация: платить поквартально равными частями (фикс / 4) для равномерного вычета из налога

## Формат вывода

```
СТРАХОВЫЕ ВЗНОСЫ ИП за {год}

Налоговый режим: {режим}
Годовой доход: {сумма} руб.

Фиксированные взносы:
  Ставка {год}: {ставка} руб.
  Пропорция (если неполный год): {расчёт}
  ИТОГО фикс: {сумма} руб.

Переменная часть (1% свыше 300К):
  База: {доход} − 300 000 = {разница} руб.
  1%: {сумма} руб.

═══════════════════════════════════════
ВСЕГО ВЗНОСОВ: {сумма} руб.
═══════════════════════════════════════

Уплачено:    {сумма} руб.
Осталось:    {сумма} руб.

Сроки:
  Фикс. часть: до 31.12.{год}
  1% свыше 300К: до 01.07.{год+1}

Рекомендуемый график (поквартально):
  Q1 (до 31.03): {фикс/4} руб.
  Q2 (до 30.06): {фикс/4} руб.
  Q3 (до 30.09): {фикс/4} руб.
  Q4 (до 31.12): {фикс/4} руб.

КБК взносов: 18210202000011000160
Оплата через ЕНП: КБК 18201061201010000510
```

## Для НПД

Если режим = нпд, выведи:
"Для плательщиков НПД страховые взносы не являются обязательными.
Вы можете уплачивать добровольные взносы на ОПС для формирования пенсионного стажа.
Добровольный взнос в 2025 году: 53 658 руб. (полный год стажа)."
