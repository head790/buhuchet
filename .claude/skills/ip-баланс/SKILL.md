Покажи свободные средства — сколько реально можно потратить.

Аргументы: $ARGUMENTS (не используются)

## Инструкции

1. Прочитай `data/profile.yaml` — налоговый режим
2. Прочитай `data/journal.csv` — доходы и расходы с начала текущего квартала
3. Прочитай `data/payments.yaml` — уже уплаченные налоги и взносы за текущий квартал

## Расчёт

```
баланс = сумма_всех_доходов - сумма_всех_расходов - сумма_всех_уплаченных_платежей
```

Это грубая оценка "сколько на счёте" на основе журнала. Если пользователь знает точный баланс — пусть скажет, используй его.

### Резерв на налоги

Рассчитай налог и взносы, которые нужно будет заплатить за текущий квартал, но ещё не уплачены:

```
# УСН 6%
резерв_налог = доход_с_начала_года * 6% - вычет_взносов - уплаченные_авансы
резерв_взносы = (53658 / 4) - уплаченные_взносы_за_квартал

# УСН 15%
резерв_налог = (доход - расходы) * 15% - уплаченные_авансы
резерв_взносы = (53658 / 4) - уплаченные_взносы_за_квартал

# НПД
резерв_налог = налог_за_текущий_месяц (4%/6%)
резерв_взносы = 0

# Патент
резерв_налог = оставшаяся_часть_патента
резерв_взносы = (53658 / 4) - уплаченные_взносы_за_квартал
```

### Свободные средства

```
свободные = баланс - резерв_налог - резерв_взносы
```

## Формат вывода

```
СВОБОДНЫЕ СРЕДСТВА на {дата}

Баланс (оценка по журналу): {сумма} руб.
Резерв на налоги:           -{сумма} руб.  ({режим}, до {дата платежа})
Резерв на взносы:           -{сумма} руб.  (до {дата платежа})
─────────────────────────────────────────
МОЖНО ТРАТИТЬ:               {сумма} руб.

Ближайший платёж: {дата} — {описание} — {сумма} руб.
```

Если свободные средства < 0, выведи предупреждение:
"⚠️ Свободных средств недостаточно. Ожидаемый дефицит: {сумма} руб. Рекомендуется ускорить получение оплаты или сократить расходы."
