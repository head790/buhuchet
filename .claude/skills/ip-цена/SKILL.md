Рассчитай цену для клиента от целевой чистой прибыли.

Аргументы: $ARGUMENTS
Формат: целевая прибыль, расходы по проекту (опционально)
Примеры:
  80000
  80000 расходы 15000
  100000 расходы 30000

## Инструкции

1. Прочитай `data/profile.yaml` — налоговый режим
2. Разбери аргументы: первое число — целевая чистая прибыль, после "расходы" — расходы по проекту (если не указаны — считай 0)

## Расчёт

Обратный расчёт: от желаемой прибыли к цене.

### УСН 6%

```
# цена = прибыль + расходы + налог + взносы_пропорц
# налог = цена * 6% (но вычитаются взносы, поэтому эффективная ставка ниже)
# Упрощённо: эффективная ставка ~6% (взносы фиксированы и уже учтены в /ip-взносы)

цена = (прибыль + расходы) / (1 - 0.06)
налог = цена * 6%
взносы_пропорц = цена / годовой_доход * годовые_взносы  # доля этого проекта в годовых взносах
```

### УСН 15%

```
# налог = (цена - расходы - взносы_пропорц) * 15%
цена = (прибыль + расходы) / (1 - 0.15) + взносы_пропорц
# Итеративный расчёт, т.к. взносы зависят от дохода
```

### НПД

```
# ставка зависит от типа клиента
цена_физ = прибыль / (1 - 0.04)   # клиент — физлицо
цена_юр = прибыль / (1 - 0.06)    # клиент — юрлицо
# Расходы не вычитаются из базы НПД
цена = прибыль_цель + расходы + налог
```

### Патент

```
# Патент фиксированный, налог не зависит от цены проекта
цена = прибыль + расходы
# Но взносы пропорциональны
```
