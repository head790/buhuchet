Добавь запись о расходе в журнал.

Аргументы: $ARGUMENTS
Формат: сумма, описание, категория (опционально)
Примеры:
  5000 Хостинг VPS хостинг
  12000 Лицензия JetBrains софт
  3000 Такси до клиента транспорт

## Категории расходов

- хостинг — серверы, домены, SSL, облачные сервисы
- софт — лицензии, подписки на ПО
- реклама — таргет, контекст, SMM
- связь — телефон, интернет
- транспорт — такси, проезд, топливо
- оборудование — техника, комплектующие
- офис — аренда, канцелярия, мебель
- прочее — всё остальное

Если категория не указана — определи автоматически по описанию. Если не удаётся определить — используй "прочее".

## Инструкции

1. Разбери аргументы: первое число — сумма, последнее слово — возможно категория (проверь по списку выше), остальное — описание
2. Прочитай `data/journal.csv`
3. Добавь новую строку:
   - date: сегодняшняя дата (YYYY-MM-DD)
   - amount: сумма
   - type: expense
   - description: описание
   - client: (пусто)
   - document_number: (пусто)
   - category: категория

## После добавления

Выведи подтверждение:

```
✓ Расход добавлен в журнал:
  Дата: {дата}
  Сумма: {сумма} руб.
  Описание: {описание}
  Категория: {категория}

Расходы за {текущий месяц}: {сумма} руб.
Расходы за {год} по категориям:
  {категория}: {сумма} руб.
  {категория}: {сумма} руб.
  ...
```
