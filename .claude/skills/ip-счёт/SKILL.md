Сгенерируй счёт на оплату.

Аргументы: $ARGUMENTS
Формат аргументов: клиент, описание услуги, сумма
Пример: ООО Ромашка Разработка сайта 100000

## Инструкции

1. Прочитай `data/profile.yaml` — реквизиты ИП (исполнителя)
2. Прочитай `data/clients.yaml` — найди клиента по имени из аргументов. Если клиент не найден — предупреди и предложи добавить
3. Прочитай `data/journal.csv` — найди максимальный document_number, присвой следующий номер
4. Сформируй номер счёта: `СЧ-{номер}/{год}` (например СЧ-6/2025)

## Формат вывода

Выведи счёт в следующем формате:

```
СЧЁТ НА ОПЛАТУ № {номер} от {дата}

Исполнитель: ИП {ФИО}
ИНН: {инн}
ОГРНИП: {огрнип}
Адрес: {адрес}
Р/с: {расчётный_счёт}
Банк: {название банка}
БИК: {бик}
К/с: {корр_счёт}

Заказчик: {полное_название клиента}
ИНН: {инн клиента}
КПП: {кпп клиента}
Адрес: {адрес клиента}

┌─────┬──────────────────────────────────┬───────────────┐
│ №   │ Наименование                     │ Сумма (руб.)  │
├─────┼──────────────────────────────────┼───────────────┤
│ 1   │ {описание услуги}                │ {сумма}       │
├─────┼──────────────────────────────────┼───────────────┤
│     │ ИТОГО:                           │ {сумма}       │
└─────┴──────────────────────────────────┴───────────────┘

Сумма прописью: {сумма прописью} рублей 00 копеек

Без НДС (ИП применяет {режим налогообложения}).

ИП {ФИО короткое} ________________
```

## Правила по режимам

- **УСН 6% / УСН 15%**: "Без НДС. ИП применяет упрощённую систему налогообложения."
- **Патент**: "Без НДС. ИП применяет патентную систему налогообложения."
- **НПД**: "НДС не облагается на основании п. 9 ст. 2 Федерального закона от 27.11.2018 № 422-ФЗ."

## После генерации

Спроси пользователя, нужно ли добавить эту сумму как доход в `data/journal.csv`. Если да — добавь строку с сегодняшней датой, суммой, type=income, описанием, клиентом и номером документа.
