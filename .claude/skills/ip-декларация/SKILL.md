Подготовь данные для заполнения налоговой декларации.

Аргументы: $ARGUMENTS
Формат: год. Пример: "2025". Без аргументов — предыдущий год.

## Инструкции

1. Прочитай `data/profile.yaml` — режим, ИНН, ОКТМО
2. Прочитай `data/journal.csv` — доходы и расходы за год поквартально
3. Прочитай `data/payments.yaml` — авансы и взносы за год

## Применимость

- **УСН 6%** — Декларация по УСН (форма по КНД 1152017), Раздел 1.1 + Раздел 2.1.1
- **УСН 15%** — Декларация по УСН, Раздел 1.2 + Раздел 2.2
- **Патент** — декларация НЕ подаётся. Выведи: "Для ПСН декларация не предусмотрена."
- **НПД** — декларация НЕ подаётся. Выведи: "Для НПД декларация не предусмотрена. Учёт ведётся в приложении «Мой налог»."

## Расчёт для УСН 6% (Раздел 2.1.1)

Поквартально нарастающим итогом:

```
Строка 110: Доходы за Q1
Строка 111: Доходы за полугодие (Q1+Q2)
Строка 112: Доходы за 9 месяцев (Q1+Q2+Q3)
Строка 113: Доходы за год (Q1+Q2+Q3+Q4)

Строка 120-123: Ставка налога: 6 (или льготная, если применимо)

Строка 130: Налог за Q1 = стр.110 × 6%
Строка 131: Налог за полугодие = стр.111 × 6%
Строка 132: Налог за 9 мес = стр.112 × 6%
Строка 133: Налог за год = стр.113 × 6%

Строка 140: Вычет взносов за Q1 (не более стр.130)
Строка 141: Вычет взносов за полугодие (не более стр.131)
Строка 142: Вычет взносов за 9 мес (не более стр.132)
Строка 143: Вычет взносов за год (не более стр.133)
```

Раздел 1.1:
```
Строка 010, 030, 060, 090: ОКТМО
Строка 020: Аванс за Q1 = стр.130 − стр.140
Строка 040: Аванс за полугодие = стр.131 − стр.141 − стр.020
Строка 070: Аванс за 9 мес = стр.132 − стр.142 − стр.020 − стр.040(050)
Строка 100: Налог за год = стр.133 − стр.143 − уплаченные авансы
```

## Расчёт для УСН 15% (Раздел 2.2)

Аналогично, но с расходами:
```
Строка 210-213: Доходы нарастающим итогом
Строка 220-223: Расходы нарастающим итогом
Строка 240-243: Налоговая база (доходы − расходы, если > 0)
Строка 250-253: Убыток (если расходы > доходов)
Строка 270-273: Налог (база × 15%)
Строка 280: Минимальный налог = стр.213 × 1%
```

## Формат вывода

```
ДАННЫЕ ДЛЯ ДЕКЛАРАЦИИ УСН за {год}
ИНН: {инн}
ОКТМО: {октмо}
Код налогового периода: 34 (год)
Код по месту нахождения: 120 (по месту жительства ИП)

═══ РАЗДЕЛ 2.1.1 (для УСН 6%) / 2.2 (для УСН 15%) ═══

{Построчные данные с номерами строк и суммами}

═══ РАЗДЕЛ 1.1 / 1.2 ═══

{Построчные данные}

Срок подачи: 25 апреля {год+1}
Срок доплаты: 28 апреля {год+1}
```
