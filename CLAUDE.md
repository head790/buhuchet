# buhuchet — AI Native ассистент для ИП в РФ

AI Native ассистент, автоматизирующий административно-финансовые операции индивидуального предпринимателя. 10 skills для Claude Code: счета, акты, договоры, расчёт налогов, взносов, дедлайны.

## Быстрый старт

1. Заполните `data/profile.yaml` своими реквизитами
2. Замените образцы клиентов в `data/clients.yaml`
3. Очистите примеры в `data/journal.csv` (оставьте заголовок)
4. Очистите `data/payments.yaml`
5. Используйте команды `/ip-*`

## Файлы данных

| Файл | Формат | Назначение |
|------|--------|-----------|
| `data/profile.yaml` | YAML | Реквизиты ИП, налоговый режим, банк |
| `data/journal.csv` | CSV | Журнал доходов и расходов |
| `data/clients.yaml` | YAML | Реквизиты контрагентов |
| `data/payments.yaml` | YAML | Уплаченные налоги и взносы |

## Доступные команды

| Команда | Назначение | Аргументы |
|---------|-----------|-----------|
| `/ip-счёт` | Счёт на оплату | клиент, описание, сумма |
| `/ip-акт` | Акт выполненных работ | клиент, описание, сумма |
| `/ip-договор` | Договор | клиент, тип, сумма, срок |
| `/ip-налог` | Расчёт налога | период (опц.) |
| `/ip-взносы` | Расчёт страховых взносов | год (опц.) |
| `/ip-квартал` | Финансовая сводка | квартал (опц.) |
| `/ip-дедлайны` | Ближайшие сроки | — |
| `/ip-декларация` | Данные для декларации | год (опц.) |
| `/ip-доход` | Добавить доход | сумма, клиент, описание |
| `/ip-расход` | Добавить расход | сумма, описание, категория |

## Поддерживаемые налоговые режимы

- **УСН 6%** (доходы) — полная поддержка
- **УСН 15%** (доходы минус расходы) — полная поддержка
- **Патент (ПСН)** — расчёт оплаты, вычет взносов
- **НПД** (самозанятость) — расчёт по ставкам 4%/6%

## Важно

- Это **не** сертифицированное бухгалтерское ПО
- Ставки и лимиты актуальны на 2025-2026; проверяйте актуальность
- С 2023 года все налоги уплачиваются через ЕНП (КБК 18201061201010000510)
- Для НПД основной инструмент — приложение «Мой налог»
- XML-декларации не генерируются — только данные для ручного ввода в ЛК ФНС
- Всегда консультируйтесь с бухгалтером по спорным вопросам
